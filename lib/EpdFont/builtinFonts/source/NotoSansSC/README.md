# Noto Sans SC (Simplified Chinese) Font Source

Place the following files here before running `scripts/gen_cjk_fonts.py`:

- `NotoSansSC-Regular.ttf`
- `NotoSansSC-Bold.ttf`

## Download

From Google Fonts: https://fonts.google.com/noto/specimen/Noto+Sans+SC

Or directly from the Noto project on GitHub:
https://github.com/notofonts/noto-cjk/releases

Select "Noto Sans CJK SC" (Simplified Chinese), download the OTF/TTF package,
and copy the Regular and Bold weights here.
